# Task Manager Dashboard

A dashboard application for managing internal tasks and team collaboration.

## Getting Started

Install dependencies and run the development server.

```bash
npm install
npm run dev
```

## Project Structure

The project follows a standard React application structure. Components are organized by feature, and state management is handled through various approaches depending on the use case.

## Features

- Task creation and management
- User dashboard
- Team collaboration tools
- Real-time updates

## Testing

Run tests using the test scripts. Different parts of the codebase use different testing approaches.

## Notes

Some parts of the codebase may use different patterns or libraries. This is intentional based on when different features were implemented.
